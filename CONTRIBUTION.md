# CONTRIBUTION DOCUMENTATION
## Terminal-Based Bible Reader and Study Tool Using Python and JSON

### Development Team Contributions

---

## 1. Overview
This document explains the individual and group contributions made in the development of the Terminal-Based Bible Reader and Study Tool. The purpose of this record is to show how each member participated, what work they handled, and how the team collaborated to complete the project.

The project required the team to design, build, test, and document a command-line Bible study tool using Python and JSON. The team used a structured process so that each member had a clear part in the project.
